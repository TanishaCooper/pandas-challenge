### District Summary

The District Summary provides a high-level overview, in a DataFrame, of the district's key metrics. You will find the following metrics and trends:

* Total Schools
* Total Students
* Total Budget
* Avg Math Score
* Avg Reading Score
* % Passing Math
* % Passing Reading
* % Overall Passing
